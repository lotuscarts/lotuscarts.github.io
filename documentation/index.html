Changing Font

Consultint uses a Font family: 'Roboto' as a default font from the Google Fonts Library. To change the Font, you will need to Edit the Bottom Link with your Custom Font link in default.css file and change the body css font-family

If you plan to use a Self Hosted font. Here is an Example for using Self Hosted Font.

    @import url('https://fonts.googleapis.com/css2?family=Poppins:wght@200;300;400;500;600;700;800;900&family=Roboto:wght@300;400;500;700;900&display=swap');

    body {
        font-family: "Poppins", sans-serif
        font-family: "Roboto", sans-serif
    }
